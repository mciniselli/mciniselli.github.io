Study I: Participants' Answers

You can find all the answers from participants in the following four tables:

  • Characteristics: contains all the characteristics of the code recommendation that practitioners indicated as important.
  • Coverage: contains the improvements that participants would like to see in terms of coverage of support (i.e., the scenarios in which IDE should provide a recommendation)
  • Accuracy: contains the improvements that participants would like to see in terms of accuracy of the code recommendation.
  • Usability: contains the improvements that participants would like to see in terms of usability.
Each table is searchable both for what concerns the answer provided by the participants as well as the "tags" we finally assigned to it, that are basically characteristics extracted from our taxonomy.
