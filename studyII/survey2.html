Study II: Survey Questions

Thanks for taking part in this anonymous survey. Please read carefully these instructions. In the survey, you will be asked to comprehend up to 30 Python code snippets having a maximum length of 50 lines of code. If you don't have experience in Python, please don't start this survey. Each code snippet is accompanied by a short explanation about what it is meant to implement. A question will be used to check if you successfully comprehended each of the snippets: the shown code snippet misses a few lines of code and we'll show you different options to complete it. If you pick the correct one, we know you comprehended the code.

For each code snippet you successfully comprehend, a few additional questions will be asked: We will show you up to four code suggestions generated by automated tools with the goal of autocompleting the part that was missing in the snippet. You are required to judge which of those (if any) would have been the most useful to complete the programming task and to explain why you believe so (through one open and two multiple-choice questions). The survey is designed to take around 15 minutes per task (time can change based on your ability in comprehending Python code). Feel free to complete how many tasks as you want.
